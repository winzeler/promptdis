│                                                                              │
│ sdk-py/examples/main.py (~60 lines)                                          │
│                                                                              │
│ Single-file CLI script. No extra deps beyond the SDK (which brings httpx +   │
│ jinja2). Uses context manager.                                               │
│                                                                              │
│ - Read PROMPTDIS_URL and PROMPTDIS_API_KEY from env                          │
│ - with PromptClient(base_url=..., api_key=...) as client:                    │
│ - Fetch by name: client.get_by_name("demo", "myapp", "greeting")             │
│ - Print prompt fields: id, name, version, model, environment, body preview   │
│ - Fetch by ID: client.get(prompt.id)                                         │
│ - Local render (Jinja2): prompt.render({"name": "Alice"})                    │
│ - Server-side render: client.render(prompt.id, {"name": "Bob"})              │
│ - Cache stats: client.cache_stats()                                          │
│ - Error handling: try client.get("nonexistent-id"), catch NotFoundError      │
│                                                                              │
│ Key API details (from sdk-py/src/promptdis/client.py):                       │
│ - Constructor: PromptClient(base_url, api_key, cache_ttl=300,                │
│ cache_max_size=1000)                                                         │
│ - render() returns a string (calls get() then prompt.render() internally)    │
│ - cache_stats() returns dict                                                 │
│ - Context manager: __enter__/__exit__ calls close()                          │
│                                                                              │
│ sdk-py/examples/README.md (~40 lines)                                        │
│                                                                              │
│ - Title, what it demonstrates                                                │
│ - Prerequisites: Python 3.11+, SDK installed (pip install -e .. or uv pip    │

│ 5. Go Example — sdk-go/examples/basic/                                       │
│                                                                              │
│ sdk-go/examples/basic/main.go (~80 lines)                                    │
│                                                                              │
│ Standard Go CLI. Zero external deps beyond the SDK. Uses                     │
│ context.Background().                                                        │
│                                                                              │
│ - promptdis.NewClient(promptdis.ClientOptions{BaseURL: ..., APIKey: ...})    │
│ - defer client.Close()                                                       │
│ - Fetch by name: client.GetByName(ctx, "demo", "myapp", "greeting") — note:  │
│ no env option in basic example                                               │
│ - Print: prompt.ID, prompt.Name, prompt.Version,                             │
│ prompt.ModelDefault("unknown"), prompt.Environment, prompt.Body[:80]         │
│ - Fetch by ID: client.Get(ctx, prompt.ID)                                    │
│ - Local render: client.RenderLocal(prompt.Body, map[string]string{"name":    │
│ "Alice"})                                                                    │
│ - Server-side render: client.Render(ctx, prompt.ID,                          │
│ map[string]interface{}{"name": "Bob"}) → result.RenderedBody                 │
│ - Cache stats: client.CacheStats() → stats.Size, stats.MaxSize, stats.TTL    │
│ - Error handling: client.Get(ctx, "nonexistent-id") with errors.As for       │
│ *promptdis.NotFoundError                                                     │
│                                                                              │
│ Key API details (from sdk-go/client.go):                                     │
│ - NewClient returns (*Client, error)                                         │
│ - GetByName uses variadic GetOption (e.g., WithEnvironment("prod"))          │
│ - Render returns (*RenderResult, error), RenderResult.RenderedBody           │
│ - CacheStats() returns CacheStats{Size, MaxSize, TTL}                        │
│ - RenderLocal takes map[string]string; Render takes map[string]interface{}   │
│                                                                              │
